# Markdown Viewer Software written in flutter


## install required libs
* [pandoc](https://pandoc.org/)
* [mermaid cli](https://github.com/mermaid-js/mermaid-cli) 

## Installation 
```bash
git clone https://github.com/TheinThanHlan/markdown_viewer.git
cd markdown_viewer/markdown_viewer
flutter build linux --release
```

## Setting Path
insert following line into your .bashrc file
```bash
export PATH=$PATH:"/PATH/TO/markdown_viewer/markdown_viewer/build/linux/x64/release/bundle/"
```


## Usage
```bash
markdown_viewer path_to_md_file
```
